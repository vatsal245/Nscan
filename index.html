NScan Network Scanner

A Simple Network Scanner written in python for
detecting devices connected to your home network

NScan Network Scanner

This tool allows you to gain information about all the connected devices on your local network.

Supported OS: Windows

NScan Port Scanner

This tool allows you to know which ports of a local device are open on a local network and which services are running.

Supported OS: Windows

The Network Scanner

Look at devices connected to your home network

How To:

1. Make sure you are connected to your home network

2. The 'scan for hosts' button should automatically detect your network details and print it in the details section.

3. If connected to a network the scan should start and devices connected to the network should appear in the 'Network Device Details' section.

The Port Scanner

Look at open ports and services running on any ip address

How To:

1. Make sure you are connected to your home network

2. The 'scan for hosts' button should automatically detect your network details and print it in the details section.

3. If connected to a network the scan should start and devices connected to the network should appear in the 'Network Device Details' section.

NOTE: Installation of WinPCap or NPcap drivers are required for the applications.
